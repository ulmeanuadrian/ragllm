Format JSON Chunkizat Acceptat

Acest sistem acceptă EXCLUSIV fișiere JSON cu chunk-uri predefinite în următorul format:

{ "chunk_0": { "metadata": "Source: document.pdf", "chunk": "Conținutul primului chunk cu informații relevante..." }, "chunk_1": { "metadata": "Source: document.pdf", "chunk": "Conținutul celui de-al doilea chunk cu alte informații..." }, "chunk_2": { "metadata": "Source: document.pdf", "chunk": "Și așa mai departe pentru fiecare chunk..." } }
Fiecare chunk va fi indexat separat pentru căutare semantică optimizată
Metadatele sunt păstrate pentru referințe sursă
Nu se acceptă alte formate de fișiere
