Format JSON Chunkizat

Acest sistem acceptă DOAR fișiere JSON cu chunk-uri predefinite în următorul format:

{ "chunk_0": { "metadata": "Source: document.pdf", "chunk": "Conținutul primului chunk..." }, "chunk_1": { "metadata": "Source: document.pdf", "chunk": "Conținutul celui de-al doilea chunk..." } }
Fiecare chunk va fi indexat separat pentru căutare semantică optimizată.
